Petfinder

O Petfinder é uma aplicação web que ajuda a encontrar pets perdidos na vizinhança. A aplicação permite a criação de usuários, autenticação, postagem de informações sobre pets perdidos e busca por pets perdidos.

Rotas

  • Criação de Usuário

    POST /api/users

    Descrição: Cria um novo usuário.

    Corpo da Requisição:

    {
    "username": "string",
    "email": "string",
    "password": "string"
}

  • Autenticação

    POST /api/auth

    Descrição: Autentica um usuário e retorna um token JWT.

    Corpo da Requisição:

    {
    "email": "string",
    "password": "string"
}

  • Cadastro de Pets Perdidos

    POST /api/pets

    Descrição: Permite que um usuário autenticado cadastre um pet perdido.

    Corpo da Requisição:

    {
    "name": "string",
    "description": "string",
    "location": "string",
    "contact": "string"
}

  • Informar sobre Pet Perdido

    POST /api/pets/:id/information

    Descrição: Permite que um usuário autenticado poste informações adicionais sobre um pet perdido.

    Corpo da Requisição:

    {
    "information": "string"
}

  • Buscar Pets Perdidos

    GET /api/pets

    Descrição: Busca por pets perdidos na vizinhança.

    Parâmetros de Consulta (Query Params):

    ?location=string&radius=number
